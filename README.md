# gorilla-cadmium
With protecting your privacy, browse anything.

## Setup
1. Clone this repo
   ```bash
   git clone https://github.com/mtripg6666tdr/gorilla-cadmium.git
   ```
2. Install dependencies
   ```bash
   npm i
   ```
3. Transpile
   ```bash
   npm run build
   ```
5. Start the server
   ```bash
   npm start
   ```
